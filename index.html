Code Quiz Challenge

Welcome to the code quize challenge. In this quiz you'll be tested on your JavaScript skills. When you're ready press the start button below to begin the quiz. When you press start the timer on the right will begin to count down from 60 seconds. The quiz will end once the timer reaches 0 or if you answer all the questions before the timer ends. For each question you get right, you will earn one point. On the otherhand, if you get a question incorrect, a second will be deducted from the timer. At the end your score will be presented with an option to enter your initials in order to record your score. If you want to view the high scores click the button on the left. Good Luck!
